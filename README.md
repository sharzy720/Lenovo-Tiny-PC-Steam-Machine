# Lenovo Tiny PC Steam Machine
Repository containing the steps and files needed to achieve the full performance of a Lenovo Tiny PC when using it as a Steam Machine with Bazzite.

---

## Steps to maximize the performance of your Lenovo Tiny PC
1. Disable BD PROCHOT by using the repository at this [link](https://github.com/fralapo/Disable-BD-PROCHOT-on-LINUX)
2. Limit the power of the GPU to the wattage of the GPUs that were originally shipped with the computer.
   - If it is an Nvidia GPU, you can use nvidia-smi to limit the power of the GPU

---

Tested devices
* Lenovo P330 Tiny
  - CPU: i7 8700T
  - GPU: Yeston 3050 6gb Single Slot
